# How to Use

Use `cargo` to build and run the executable:

```
$ cargo run -- <PATH>
```

where `<PATH>` is the path to the Quake 3 Arena log file:

```
$ cargo run -- qgames.log
```

# Testing

Unit tests can be run with:

```
$ cargo test
```
